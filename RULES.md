# ManTRPG Core Rules

## 기본

- 적은 층마다 항상 1명이다.
- 보스몹은 없다.
- 전투는 턴제다.
- 맵은 고정된 2D 격자 맵이다.
- 플레이어와 적은 맵 좌표를 가진다.
- 전투 로그와 하단 맵을 함께 표시한다.

## 스탯

기본 스탯:
- 힘
- 민첩
- 체력
- 지능
- 지혜
- 외모

스탯 값이 주사위 최대값이다.
예: 힘 12 → 1d12

## 전투

- 한 턴에 기본 행동 1회.
- 공격, 방어, 이동, 마법, 관찰, 자유 행동 가능.
- 주사위는 코드가 굴린다.
- 판정 결과는 로그에 표시한다.

## 마법

- 마법은 즉시 발동한다.
- 기본적으로 준비-발동 2단계 구조를 쓰지 않는다.
- MP가 부족하면 발동하지 않는다.
- MP 차감은 코드가 처리한다.
- 마법 효과는 맵에 표시한다.

## 보상

층 클리어 보상:
- 스킬 초기화권 5%
- 무공서 10%
- 마법서 25%
- 추가 코인 +1 40%
- 추가 코인 +2 20%

보상은 2개 생성 후 1개 선택한다.
보상은 AI가 생성하지 않고 코드가 생성한다.

## 심상세계 순서

1. 보상 획득
2. 스탯 분배
3. 스킬 생성
4. 마법서 관련 행동
5. 상점 이용
6. 다음 층 진입 여부 선택

## MVP 파생 수치 공식

- 최대 HP = 20 + 체력
- 최대 MP = 4 + 지능
- 지혜 기반 MP 회복은 추후 구현한다.


## 초기 스탯 분배

- 기본 스탯은 힘, 민첩, 체력, 지능, 지혜, 외모다.
- 각 스탯은 최소 1에서 시작한다.
- 초기 분배 가능 포인트는 48이다.
- 초기 총합은 54다.
- 1레벨 기준 직접 분배 최대치는 21이다.
- 초기 스탯 포인트를 모두 사용해야 1층 전투를 시작할 수 있다.

## MVP 마법서 규칙
- 현재 MVP에서는 보유 마법서를 기초 마법서로 처리한다.
- 기초 마법서 습득 판정은 1d50이다.
- 성공 조건은 주사위 결과 < 지혜다.
- 지혜 50 이상이면 자동 성공한다.
- 실패해도 마법서는 사라지지 않는다.
- 성공 시에만 마법서 1개를 소모하고 1서클 고정 마법 풀에서 마법을 배운다.
- 심상세계마다 기본 시도는 1회다.
- 코인 1개로 추가 시도할 수 있다.
