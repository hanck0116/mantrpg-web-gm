# ManTRPG Core Rules

## 기본

- 적은 층마다 항상 1명이다.
- 보스몹은 없다.
- 전투는 턴제다.
- 맵은 고정된 2D 격자 맵이다.
- 플레이어와 적은 맵 좌표를 가진다.
- 전투 로그와 하단 맵을 함께 표시한다.

## 스탯

기본 스탯:
- 힘
- 민첩
- 체력
- 지능
- 지혜
- 외모

스탯 값이 주사위 최대값이다.
예: 힘 12 → 1d12

## 전투

- 한 턴에 기본 행동 1회.
- 공격, 방어, 이동, 마법, 관찰, 자유 행동 가능.
- 주사위는 코드가 굴린다.
- 판정 결과는 로그에 표시한다.

## 마법

- 마법은 즉시 발동한다.
- 기본적으로 준비-발동 2단계 구조를 쓰지 않는다.
- MP가 부족하면 발동하지 않는다.
- MP 차감은 코드가 처리한다.
- 마법 효과는 맵에 표시한다.

## 보상

층 클리어 보상:
- 스킬 초기화권 5%
- 무공서 10%
- 마법서 25%
- 추가 코인 +1 40%
- 추가 코인 +2 20%

보상은 2개 생성 후 1개 선택한다.
보상은 AI가 생성하지 않고 코드가 생성한다.

## 심상세계 순서

1. 보상 획득
2. 스탯 분배
3. 스킬 생성
4. 마법서 관련 행동
5. 상점 이용
6. 다음 층 진입 여부 선택
