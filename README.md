# ManTRPG Web GM

ManTRPG Web GM은 ManRPG v18 FINAL 규칙을 기반으로 한 모바일 우선 PWA 턴제 TRPG입니다.

## 목표

- 모바일에서 플레이 가능
- 무료 배포 가능
- 서버 없이 작동
- AI 없이도 플레이 가능
- 규칙은 코드로 강제
- 하단 전술 맵 표시
- 적은 항상 1명
- 고정 맵 사용
- 간단한 UI 유지

## 모바일 실행 방법

- 모바일 브라우저에서 접속한다.
- 새 게임을 누른다.
- 초기 스탯을 분배한다.
- 저장 버튼을 누르면 현재 상태가 기기에 저장된다.
- 불러오기 버튼으로 이어서 할 수 있다.



## 모바일 화면 구조

- 상단에는 HP, MP, 코인, 층, 턴만 간단히 표시된다.
- 상세 정보 버튼을 누르면 인벤토리, 현재 단계, 스킬/마법 수, 적 정보, 앱 버전을 확인할 수 있다.
- 중앙에는 전투 로그와 현재 단계 안내가 표시되어 다음 행동을 쉽게 확인할 수 있다.
- 하단에는 모바일에서 누르기 쉬운 행동 버튼과 7x7 전술 맵이 표시된다.

## 저장 데이터 안내

- 게임 상태는 브라우저 localStorage에 저장된다.
- 앱을 다시 열면 저장 데이터가 있다는 안내가 표시된다.
- 이어서 하려면 불러오기를 누른다.
- 새 게임을 누르면 기존 로컬 저장 데이터가 삭제된다.
- 중요한 저장은 저장 내보내기로 JSON 파일을 따로 보관하는 것을 권장한다.
- 저장 가져오기는 같은 프로젝트의 JSON 저장 파일만 사용한다.

## 테스트 모드

- 테스트 모드는 배포 전 기능 점검용이다.
- 기본값은 OFF다.
- 실제 배포 시 app.js의 ENABLE_TEST_MODE는 false로 유지한다.
- 테스트 모드에서는 코인, 마법서, 스킬, 마법을 임시 지급할 수 있다.
- 테스트 모드에서는 적 HP를 1로 만들거나 현재 층 클리어를 강제로 실행할 수 있다.
- 일반 플레이 밸런스 확인 시에는 테스트 모드를 사용하지 않는다.
- 배포 전에는 테스트 모드를 닫은 상태에서 일반 플레이 흐름을 한 번 이상 확인한다.
- 테스트 모드 열림/활성 상태는 저장 데이터에 포함하지 않는 UI 전용 상태다.

## PWA 설치 방법

Android Chrome:
- 사이트 접속
- 메뉴
- 홈 화면에 추가 또는 앱 설치

iPhone Safari:
- 공유 버튼
- 홈 화면에 추가

## 무료 배포 방법

Cloudflare Pages 또는 GitHub Pages에 정적 파일로 배포할 수 있다.
서버, 외부 DB, 유료 API가 필요 없다.
자세한 배포 절차는 DEPLOY.md를 참고한다.

## 추가 문서

- 배포 절차: DEPLOY.md
- 최종 점검표: FINAL_CHECKLIST.md
- 릴리즈 노트: RELEASE_NOTES.md

## 주의

- 저장 데이터는 사용자의 브라우저 localStorage에 저장된다.
- 브라우저 데이터를 삭제하면 저장 데이터도 삭제될 수 있다.
- 현재 버전은 MVP이며 모든 기능은 로컬에서 작동한다.

## 배포 전 테스트 체크리스트

- 새 게임을 눌렀을 때 INITIAL_STAT 단계가 표시되는지 확인
- 추천 분배가 작동하는지 확인
- 스탯 분배 완료 후 BATTLE로 넘어가는지 확인
- 공격, 방어, 이동, 관찰이 작동하는지 확인
- 적을 처치하면 보상 선택으로 넘어가는지 확인
- 보상 선택 후 스탯 분배로 넘어가는지 확인
- 스킬 생성/생략 단계가 작동하는지 확인
- 마법서 단계가 작동하는지 확인
- 상점 단계가 작동하는지 확인
- 다음 층 진입이 작동하는지 확인
- 저장/불러오기가 작동하는지 확인
- 저장 내보내기/가져오기가 작동하는지 확인
- 모바일에서 화면이 너무 작게 깨지지 않는지 확인
- PWA 홈 화면 추가가 가능한지 확인
