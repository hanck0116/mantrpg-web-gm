# ManTRPG Web GM

ManTRPG Web GM은 ManRPG v18 FINAL 규칙을 기반으로 한 모바일 우선 PWA 턴제 TRPG입니다.

## 목표

- 모바일에서 플레이 가능
- 무료 배포 가능
- 서버 없이 작동
- AI 없이도 플레이 가능
- 규칙은 코드로 강제
- 하단 전술 맵 표시
- 적은 항상 1명
- 고정 맵 사용
- 간단한 UI 유지

## 모바일 실행 방법

- 모바일 브라우저에서 접속한다.
- 새 게임을 누른다.
- 초기 스탯을 분배한다.
- 저장 버튼을 누르면 현재 상태가 기기에 저장된다.
- 불러오기 버튼으로 이어서 할 수 있다.

## PWA 설치 방법

Android Chrome:
- 사이트 접속
- 메뉴
- 홈 화면에 추가 또는 앱 설치

iPhone Safari:
- 공유 버튼
- 홈 화면에 추가

## 무료 배포 방법

Cloudflare Pages 또는 GitHub Pages에 정적 파일로 배포할 수 있다.
서버, 외부 DB, 유료 API가 필요 없다.

## 주의

- 저장 데이터는 사용자의 브라우저 localStorage에 저장된다.
- 브라우저 데이터를 삭제하면 저장 데이터도 삭제될 수 있다.
- 현재 버전은 MVP이며 모든 기능은 로컬에서 작동한다.
