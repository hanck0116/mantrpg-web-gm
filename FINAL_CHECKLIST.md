# ManTRPG Web GM 최종 점검표

## 1. 새 게임 확인

- [ ] 새 게임 버튼을 누르면 INITIAL_STAT 단계가 표시된다.
- [ ] 추천 분배 버튼이 작동한다.
- [ ] 스탯 포인트를 모두 사용해야 전투가 시작된다.

## 2. 전투 확인

- [ ] 공격이 작동한다.
- [ ] 방어가 작동한다.
- [ ] 방향 이동이 작동한다.
- [ ] 접근 이동이 작동한다.
- [ ] 관찰이 작동한다.
- [ ] 적은 항상 1명만 등장한다.
- [ ] 적 HP가 0이 되면 보상 선택으로 넘어간다.

## 3. 심상세계 확인

- [ ] 보상 후보 2개가 표시된다.
- [ ] 보상 선택 후 스탯 분배로 넘어간다.
- [ ] 스탯 분배 후 스킬 생성 또는 생략이 진행된다.
- [ ] 마법서 단계가 진행된다.
- [ ] 상점 단계가 진행된다.
- [ ] 다음 층 진입 단계가 진행된다.

## 4. 저장 확인

- [ ] 저장이 작동한다.
- [ ] 불러오기가 작동한다.
- [ ] 저장 삭제가 작동한다.
- [ ] 저장 내보내기가 작동한다.
- [ ] 저장 가져오기가 작동한다.

## 5. PWA 확인

- [ ] 모바일 브라우저에서 접속된다.
- [ ] 홈 화면 추가가 가능하다.
- [ ] 앱처럼 실행된다.
- [ ] service worker 캐시가 정상 등록된다.
- [ ] 수정 후 캐시 버전 변경이 적용된다.

## 6. 배포 전 확인

- [ ] ENABLE_TEST_MODE가 false인지 확인한다.
- [ ] 테스트 모드 버튼이 화면에 보이지 않는지 확인한다.
- [ ] 외부 API, 서버, DB 연결이 없는지 확인한다.
- [ ] Cloudflare Pages 또는 GitHub Pages 정적 배포로 실행되는지 확인한다.
